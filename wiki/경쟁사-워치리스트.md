# 경쟁사 워치리스트

> [!takeaway] 내 관점 핵심
> - 추적 대상은 고정이 아니다. **계속 늘린다.** 새 플레이어가 우리 카테고리(헬스식품·헬스장비·여성 웰니스)에 들어오면 여기 후보로 올리고, 검증되면 [[뉴스검색-루틴|수집 루틴]](`scripts/config.json`)에 쿼리를 추가해 추적을 시작한다.
> - WELLNESS(여성·일상·루틴) 관점에선 **일화(팅글)** 같은 일상 웰니스 브랜드가 기존 헬스 경쟁사보다 더 중요한 벤치마크일 수 있다.

## 후보 큐 (추적 검토 대상)
> 상태: 🆕후보 / 🔍검토중 / ✅추적시작 / ❌제외
> 뉴스·블로그에서 새 플레이어가 잡히면 여기 쌓고, 카테고리가 분명하면 위 '현재 추적 중'으로 자율 승격한다.

| 브랜드 / 플레이어 | 카테고리 | 왜 주목 | 상태 | 포착일 |
|---|---|---|---|---|
| _(비어 있음 — 다음 회차 발굴 대기)_ | | | | |

## 추가 기준 (1개라도 맞으면 후보로 적재)
- 우리 카테고리(헬스식품 / 헬스장비 / 여성·일상 웰니스)에서 **반복 노출**되는 브랜드
- **WELLNESS 타겟(여성·일상·루틴)**과 겹치는 신규 브랜드
- 새 **원료·제형·브랜딩**으로 화제가 된 플레이어

## 추적 시작하는 법
1. 후보 검증 (실제 경쟁/벤치마크 가치 확인) → 상태 ✅추적시작
2. `scripts/config.json`의 `brands` 배열에 `{ "name", "query", "tag": "경쟁" }` 추가
3. 다음 수집부터 `raw/`에 잡힘 → [[00-모니터링-목차]] 추적 표·일별 로그에 반영
