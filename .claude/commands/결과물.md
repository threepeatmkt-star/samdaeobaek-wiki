description: wiki에 누적된 백데이터로 사고해 업무 결과물(브리핑·기획안·비교표·카피)을 outputs/에 생성
---

`$ARGUMENTS`에 적힌 결과물을 만들어줘. 비어 있으면 어떤 결과물을 원하는지 먼저 물어봐.

`CLAUDE.md`의 "④ 결과물 만들기" 규칙에 따라:

1. **근거 위키 읽기**: `wiki/index.md`로 관련 페이지를 찾아 읽는다. (회의록 같은 외부 입력이 아니라, 계속 누적한 시장·경쟁·트렌드 백데이터로 직접 사고한다.)
2. **작성**: `outputs/`에 마크다운으로 작성. 파일명 `YYYY-MM-주제.md`. 맨 위에 **생성일 + 근거 위키([[링크]])** 명시(스냅샷).
3. **삼대오백/MXSSIVE/WELLNESS 관점**으로 실무에서 바로 쓸 수 있게. 광고 카피류는 [[건기식-광고규제-과징금강화]] 가드레일 준수.
4. **등록**: `outputs/00-결과물-목차.md`와 `wiki/log.md`에 한 줄 기록.
5. **요약**: 만든 결과물 경로 + 핵심 1~2줄 + 💡 추가로 뽑으면 좋을 결과물 제안.

기본 유형: 시장·경쟁 브리핑 / 경쟁사 비교표 / 신제품 기획안 초안(WELLNESS) / 마케팅 카피 초안 / 실행 아이디어 리스트.

⚠️ 네이버 API 키 등 시크릿은 본문에 절대 쓰지 마.
