# 📥 raw/ — 자료를 여기에 그냥 떨어뜨리세요

이 폴더는 **원본 자료 드롭존**입니다. **하위 폴더를 만들지 마세요. 파일을 바로 여기에 넣으면 됩니다.**

- ✅ **당신**: 파일을 이 폴더에 직접 넣습니다. (기사·PDF·메모·캡처 텍스트·회의록 등 무엇이든, 형식 무관)
- 📖 **Claude**: 읽기만 합니다. raw/는 절대 수정·삭제하지 않습니다.
- 🗂️ **분류는 여기서 하지 않습니다.** 정리·분류는 모두 `../wiki/`(특히 `index.md` 카테고리)에서 일어납니다.

## 넣는 2가지 방법
1. **파일 드롭** — 이 폴더에 파일을 떨어뜨린 뒤 Claude에게 "정리해줘"
2. **채팅 붙여넣기** — 내용을 붙여넣으면 Claude가 여기에 파일로 저장 후 정리

## 권장 파일명 (선택)
`YYYY-MM-DD_주제.확장자` — 예: `2026-06-17_경쟁사-프로틴-신제품.txt`
폴더 없이도 날짜·주제로 정렬·추적이 됩니다.

> 모든 자료는 이 폴더 **최상단에 평평하게(flat)** 쌓입니다. 폴더로 나누지 않습니다.
